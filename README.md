# BRAIAN OS

BRAIAN OS es un sistema financiero personal desarrollado para Braian Montoya.

## Objetivos

- Controlar ingresos y gastos.
- Administrar ahorros.
- Llevar seguimiento de deudas.
- Planificar la compra de un vehículo.
- Mejorar hábitos financieros.
- Funcionar como una aplicación instalable (PWA).

## Estado del proyecto

Versión: 0.1.0

## Funciones

- Dashboard financiero
- Registro de gastos
- Registro de ingresos
- Ahorros
- Objetivos
- Hábitos
- Funcionamiento offline
- Instalable en iPhone y Android

## Tecnologías

- HTML5
- CSS3
- JavaScript
- LocalStorage
- Progressive Web App
